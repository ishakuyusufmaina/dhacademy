JupyterLite Integration Demo

Running a full Python environment directly inside your browser via WebAssembly (Pyodide).

Quick Start Guide

  1. Wait a few seconds for the JupyterLite environment to initialize in the workspace.
  2. Under the Notebook section on the right, click Python (Pyodide) to open a new notebook.
  3. Type some standard Python code in a cell, for example: 
    print("Hello from the browser!")
  4. Press Shift + Enter to execute the cell.
